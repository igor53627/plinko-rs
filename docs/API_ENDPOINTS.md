# API Endpoints

This repository does not expose a network API. It provides CLI binaries and file-based outputs.

## Interfaces

- CLI binaries: `plinko`, `plinko_hints`, `bench_gpu_hints`, `cost_estimate`.
- File outputs: `database.bin`, mappings, hints.

For usage, see `README.md`, `docs/DEPLOYMENT.md`, and `docs/hint_generation.md`.
